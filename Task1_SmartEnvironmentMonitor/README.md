Task 1 – Smart Environmental Monitor

Objective

Develop a real-time environmental monitoring system using ESP32 and DHT22 sensor.

Features

* Temperature Monitoring
* Humidity Monitoring
* Environmental Status Detection
* Min/Max Value Tracking
* Non-Blocking Timing Logic

Components Used

* ESP32
* DHT22 Sensor

Software Used

* Arduino IDE
* Wokwi Simulator

Output

The system continuously measures and displays temperature and humidity values through the Serial Monitor.

Learning Outcomes

* Sensor Interfacing
* Real-Time Data Acquisition
* Embedded Systems Programming
