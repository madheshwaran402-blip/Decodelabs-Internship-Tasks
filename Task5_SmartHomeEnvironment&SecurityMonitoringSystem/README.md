Task 5 – Smart Home Environment & Security Monitoring System

Objective

Develop an IoT mini project that combines environmental monitoring and security detection.

Features

* Temperature Monitoring
* Motion Detection
* Intruder Alert System
* High Temperature Warning
* Critical Alert Mode
* LED and Buzzer Notifications

Components Used

* ESP32
* DHT22 Sensor
* PIR Motion Sensor
* LEDs
* Buzzer

Software Used

* Arduino IDE
* Wokwi Simulator

Output

The system monitors environmental conditions and security events while generating appropriate alerts.

Learning Outcomes

* Multi-Sensor Integration
* IoT System Design
* Real-Time Monitoring
* Embedded Decision Making
