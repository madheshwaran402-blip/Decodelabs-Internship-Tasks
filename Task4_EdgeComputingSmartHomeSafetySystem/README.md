Task 4 – Edge Computing Smart Home Safety System

Objective

Build a smart home safety system capable of motion detection and emergency response.

Features

* PIR Motion Detection
* Hardware Interrupts
* Emergency Override Logic
* Alarm System
* Safety Monitoring

Components Used

* ESP32
* PIR Sensor
* Potentiometer (Gas Sensor Simulation)
* LEDs
* Buzzer

Software Used

* Arduino IDE
* Wokwi Simulator

Output

The system detects motion and activates emergency alerts during abnormal conditions.

Learning Outcomes

* Interrupt Service Routines (ISR)
* Edge Computing
* Safety-Critical Design
