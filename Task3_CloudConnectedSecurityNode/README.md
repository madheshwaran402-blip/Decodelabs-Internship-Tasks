Task 3 – Cloud Connected Security Node

Objective

Develop a cloud-connected security monitoring system using ESP32 and ultrasonic sensing.

Features

* Distance Measurement
* Intruder Detection
* Wi-Fi Connectivity
* Cloud Dashboard Integration
* Real-Time Telemetry

Components Used

* ESP32
* HC-SR04 Ultrasonic Sensor
* Adafruit IO Dashboard

Software Used

* Arduino IDE
* Wokwi Simulator
* Adafruit IO

Output

The system measures distance and sends real-time data to a cloud dashboard for remote monitoring.

Learning Outcomes

* IoT Telemetry
* Cloud Integration
* Wireless Communication
