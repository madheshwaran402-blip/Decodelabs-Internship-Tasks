Task 2 – Automated Irrigation Controller

Objective

Design an automated irrigation system that monitors soil moisture and controls irrigation automatically.

Features

* Soil Moisture Monitoring
* Moisture Percentage Calculation
* Automatic Pump Control
* Dry/Wet Soil Detection
* Hysteresis Control Logic

Components Used

* ESP32
* Potentiometer (Soil Moisture Simulation)
* LED (Pump Simulation)

Software Used

* Arduino IDE
* Wokwi Simulator

Output

The system activates irrigation when moisture levels fall below a predefined threshold.

Learning Outcomes

* ADC Reading
* Automation Systems
* Threshold-Based Decision Making
